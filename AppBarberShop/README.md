# 💈 BarberBooking

> A Full Stack web application for managing barbershop services and appointments, built with **Spring Boot** and **Angular**.

---

## 📖 About

BarberBooking is a Full Stack web application designed to simplify the management of barbershop services and customer appointments.

The project has been developed to simulate a real-world business application, following modern software engineering practices and a layered architecture.

The **backend is fully implemented** using **Spring Boot**, exposing RESTful APIs for authentication, service management and booking management.

The **frontend is currently under development** using **Angular** and will provide a modern, responsive user interface that consumes the backend APIs.

This project represents one of my main portfolio applications and has been developed to strengthen my Full Stack development skills.

## 🎯 Project Goals

The application has been designed to:

- Manage barbershop services
- Manage customer bookings
- Implement secure authentication
- Apply role-based authorization
- Develop a complete REST API
- Integrate a modern Angular frontend

The objective is to build a complete Full Stack application that reflects real-world development practices.
